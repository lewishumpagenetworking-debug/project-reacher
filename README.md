# Project Reacher Web App

A lightweight mobile-friendly physique dashboard.

## How to use
1. Open `index.html` in a browser.
2. Add the folder to Netlify, Vercel, GitHub Pages, or your own hosting.
3. On iPhone, open the hosted URL in Safari and choose:
   Share > Add to Home Screen.

## Data
The app saves data locally in the browser using localStorage.
Use the Export button weekly to back up your data.
